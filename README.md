# Review Confidence Ranker

Review Confidence Ranker compares options by average rating and review count, then ranks them with conservative and balanced scoring methods.

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm test
npm run build
```

## GitHub Pages

The repository includes a GitHub Actions workflow that builds the app and deploys the `dist` output to GitHub Pages on pushes to `main`.
